My Works

Chat Server - C# and WPF

A chat server hosted on a local server allowing for user creation and login. Users can have public and private chats, the server also allows for users to play rock paper scissors with up to 2 players.

Car AI behaviour demo - C++

Using the framework given by the university, AI steering behaviours such as seeking, pursuing and object avoidance were given to the car to allow it to drive autonomously. These behaviours were managed through a state machine with given priorities to each behaviour. Another mode changes the demo to show case the car using A* pathfinding to traverse way points to either get a speed boostrap pick up a passenger or get fuel depending on the cars conditions.

Dungeon 4 - C++ and Unreal Engine

A demo for a dungeone crawler style game with a focus on building the foundations in C++ and then implementing them with unreals blueprints system. The main focuses were on the animations and UI where events were used to improve effeciency of the game over using a basic loop system to check for changes such as a change in the players health or when activating the hitbox of the sword during an attack.

Sky Runner - C++ and OpengGL

An endless runner style game created using self created models and objects. Object-oriented programming was used to set up various elements of the game such as the game objects, an object loader and a game manager. The object loader class was set up to read a text file containing model data so the game objects could be seen within the game.

Coin Hero - C++ and S2DL

A pacman inspired game using the universities S2DL framework. Object-oriented programming was used throughout the project due to the benefits that come with inheritance. The best example of this are all the game objects including items, the player and enemies. They all inherit from one class as the have common attributes that they all need such as collision boxes, positions and size.
